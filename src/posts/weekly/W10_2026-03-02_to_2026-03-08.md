## 📝 四、技术沉淀

### 4.1 技术突破

- **运行时环境类型隔离**: 多环境（Browser DOM 与 Cloudflare Edge Worker）混合项目下的 TS 类型干净剥离。

### 4.2 问题解决

**问题**: Cloudflare Workers 文件在普通的 Next.js / React 架构中抛出满屏的类型报错，引发 IDE 解析障碍。
**解决方案**: 为 workers 目录单独创建并配置独立的 `tsconfig.json`，指定 `types: ["@cloudflare/workers-types"]`，并在根目录 `tsconfig.json` 配置中将其 `exclude`。
**经验**: 在处理全栈/Serverless项目时，不同目录实质上跑在不同的 JavaScript 运行时中，因此必须分发对应环境的上下文（Context）。

### 4.3 方法论沉淀

研发推进从"硬刚问题"升级为"双AI模型并行"的方法论。
